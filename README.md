# smartpsslite
Tentang Software

Smart PSS Lite Official Dahua

Smart PSS Lite merupakan software monitoring dan management berbasis PC dari Dahua Technology yang dirancang untuk memudahkan pengelolaan perangkat keamanan seperti CCTV, NVR/XVR, Access Control, Video Intercom, dan Attendance dalam satu platform ringan dan mudah digunakan.

Software ini cocok digunakan untuk kebutuhan monitoring rumah, kantor, toko, gudang, sekolah, hingga skala bisnis kecil dan menengah (SMB). Smart PSS Lite mendukung koneksi melalui IP Address maupun P2P/Serial Number sehingga memudahkan akses perangkat secara lokal maupun remote.

Cara Install
Download installer Smart PSS Lite.
Jalankan file installer.
Klik Next hingga proses instalasi selesai.
Login atau buat password administrator lokal.
Tambahkan device menggunakan:
IP Address
Domain
Serial Number (P2P)

Cara Menambahkan Device
Buka menu Device.
Klik Add Device.
Pilih metode koneksi:
IP/Domain
SN (P2P)
Masukkan:
IP / Serial Number
Username
Password
Klik Add.

Catatan Penting
Pastikan perangkat dan PC berada dalam jaringan yang sama saat setup awal.
Gunakan password device yang benar untuk menghindari status offline.
Untuk koneksi P2P, pastikan fitur P2P pada device sudah aktif.
Beberapa fitur advanced mungkin hanya tersedia pada Smart PSS versi full/non-lite.
